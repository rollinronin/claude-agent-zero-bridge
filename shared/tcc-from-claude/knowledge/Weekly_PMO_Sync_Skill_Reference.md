# Weekly PMO Sync — Skill Reference
_Added: 2026-04-10 (Session 13)_

A Claude Desktop skill (weekly-pmo-sync) that automates weekly sync of TCC IS project data from Hub RAID log to PMO org-wide SharePoint lists. Two lists: Risks and Issues, Status Report. PMO IDs: EXP=222, NG911=193, VuWall=11138. Key rules: no named individuals, Med not Medium, RAID vs RAIDD. Skill files: SKILL.md, references/field-mapping.md, references/gotchas.md.
